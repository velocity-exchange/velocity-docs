# Contributing to Drift Protocol

Interested in contributing to the Drift Protocol? There are multiple ways to make valuable contributions, from documentation to code and more. Below are some of the primary avenues you can explore.

## Documentation
Raise issues or submit Pull Requests at the [Drift Documentation Repository](https://github.com/velocity-exchange/drift-protocol-v2-docs).

## Bot Code

### Keeper Bots
- Repository: [keeper bots](https://github.com/velocity-exchange/keeper-bots-v2)
- Example: ```filler-lite``` is a recent effort to make running a keeper bot more accessible for lower-performing RPCs. [Learn More](https://github.com/velocity-exchange/keeper-bots-v2/blob/master/src/bots/fillerLite.ts)

### Trading Bots
- JIT Proxy Program: [Repository](https://github.com/velocity-exchange/jit-proxy)
- JIT Maker: [Source Code](https://github.com/velocity-exchange/keeper-bots-v2/blob/master/src/bots/jitMaker.ts)

## SDKs
Add new features or improve code quality and maintainability to our various SDKs.

| SDK | GitHub | Autogenerated Docs |
|---|---|---|
| TypeScript (`@drift-labs/sdk`) | [drift-labs/protocol-v2/sdk](https://github.com/velocity-exchange/protocol-v2/tree/master/sdk) | [TypeScript SDK docs](https://velocity-exchange.github.io/protocol-v2/sdk/) |
| Python (`driftpy`) | [drift-labs/driftpy](https://github.com/velocity-exchange/driftpy) | [driftpy docs](https://velocity-exchange.github.io/driftpy/) |
| Rust (`drift-rs`) | [drift-labs/drift-rs](https://github.com/velocity-exchange/drift-rs) | [docs.rs drift-rs](https://docs.rs/drift-rs/) |

## Composing Programs on Top of Drift
- Example: [Drift Vaults](https://github.com/velocity-exchange/drift-vaults)
- Feature Requests: Tokenization of the insurance fund stake has been requested.

## Main Program Contributions
- Expanding the test suite (both TypeScript and Cargo tests) at the [Main Program Repository](https://github.com/velocity-exchange/protocol-v2/tree/master/programs/drift).

Feel free to pick a project or issue that interests you and start contributing!
