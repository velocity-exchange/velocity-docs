# Keeper Bots

Keeper bots keep Velocity operational by performing automated actions as off-chain agents — matching orders, triggering conditional orders, and liquidating unhealthy positions. All four reference bots ship from [`apps/keeper-bots-v2`](https://github.com/velocity-exchange/velocity-v1/tree/master/apps/keeper-bots-v2) in the `velocity-v1` monorepo; see [Trading Automation](/developers/trading-automation) for shared setup (wallet, RPC provider, config, and running the app).
