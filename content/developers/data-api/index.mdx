# Data API Playground

At a high level, the Velocity Data API is the backbone for reading historical and real-time protocol data so that apps, analytics, and integrations don't need to index the chain themselves. It follows the OpenAPI spec.

<Callout type="warning">
**Provisional endpoint.** Velocity's hosted Data API host is not yet finalized. This page uses `data.velocity.exchange` as a placeholder pattern (mirroring Drift's `data.api.drift.trade`) — confirm the real host and OpenAPI spec URL before shipping against it.
</Callout>

<Callout type="info">
All endpoints are relative to `https://data.velocity.exchange`.
</Callout>
